US Police Shootings Events

Hello, Jeff or anyone else who is reading these words,

Nice to have you here! This web application is based on a data collection of the police shootings happened from Jan. 1, 2015 to Jun. 15, 2020.
For each shooting event, it includes some details about the instant situation when the shooting happened, some very basic personal information
of the dead, as well as the shooting's location in terms of cities and states.
For a brief introduction of the web app itself, there are two core pages of it:
This page where you can explore the database by shooting case criteria and search using ids and this page where you can see some graphs and investigate your state of interest.
There's also a page which is used to show the full result of a searching utility.
However, it does not show anything unless you reach there from the other page. So I recommend starting from the first page here.
Hopefully you'll enjoy exploring this web application!
